Automatically updates product stock status on a specified date.

== Description ==

Auto Product Restock for Woo is a powerful tool for WooCommerce store owners to automatically update the stock status of their products based on a specified date. With this plugin, you can ensure that your products are restocked on time, without any manual intervention.

**Features:**
* Automatically update product stock status on a specified date.
* Option to specify restock quantity for products that manage stock.
* Customizable settings to define the cron job schedule.
* Display the expected restock date on the product page.
* Multilingual support.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce->Auto Restock Settings screen to configure the plugin.

== Frequently Asked Questions ==

= How do I set a restock date for a product? =

Go to the product edit page, and in the 'Inventory' tab, you will find the fields to set the restock date and restock quantity.

= Can I customize the cron job schedule? =

Yes, you can set the number of hours between each cron job run in the plugin settings page under WooCommerce->Auto Restock Settings.

= Does this plugin support multilingual sites? =

Yes, the plugin includes support for multiple languages. You can add your own translations by editing the `.po` files in the `languages` directory.
